Personal Relay · Cloudflare Pages · D1

个人自用万能中转网关

面向个人维护者的统一网络出口。它把 OpenAI、Gemini、npm、Docker Registry 等 HTTP 或流式请求汇聚到一个干净域名，同时提供一个可响应、可登录、可维护的后台。

进入管理后台 Cloudflare Pages

当前系统能力

适合处理 REST、SSE、上传下载、镜像拉取与包管理等 HTTP 类请求；路由配置与管理员账户均存储在 D1 中。

响应式后台 /admin
认证方式 账号 + 密码
存储引擎 D1 Database

推荐挂载方式

常见服务建议按路径前缀转发。这样你只需要替换基础地址，不需要额外修改原始请求头、请求体和调用方式。

模型 API

OpenAI 与 Gemini 这类模型接口，适合保留客户端原始鉴权与请求结构，只在网关层替换域名和路径前缀。

/openai → https://api.openai.com
/gemini → https://generativelanguage.googleapis.com

包管理与镜像

npm 与 Docker Registry 也可以按路径映射，这样客户端只需切换到你的网关域名，后续下载与认证仍从网关继续中转。

/npm → https://registry.npmjs.org
/docker → https://registry-1.docker.io

如果你已经完成部署，可以直接进入后台添加或调整路由；如果尚未部署，建议先完成 Pages 与 D1 绑定，再初始化管理员账户。
