Personal Relay · Cloudflare Pages · D1

个人自用万能中转网关

面向个人维护者的统一网络出口。它把 OpenAI、Gemini、npm、Docker Registry 等 HTTP/S 服务汇成一个干净域名，同时提供一个响应式后台，用来登录、管理、启停和更新所有转发路由。

进入管理后台 Cloudflare Pages

当前系统能力

适合做 REST、SSE、上传下载、镜像拉取、包管理 API 等 HTTP 类请求中转。所有路由数据与后台账户均存储在 D1 中。

响应式后台 /admin
认证方式 账号 + 密码
存储引擎 D1 Database

推荐挂载方式

OpenAI / Gemini 等 API 服务建议使用路径前缀转发，便于本地 SDK 直接改 base URL。

/openai → https://api.openai.com
/gemini → https://generativelanguage.googleapis.com

npm / Docker Registry 这类镜像场景也可按路径映射，后续在客户端只需换成你的网关域名。

/npm → https://registry.npmjs.org
/docker → https://registry-1.docker.io
